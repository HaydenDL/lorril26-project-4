# Smart Schema Generator — infer-schema

Infer a SQL CREATE TABLE statement from a CSV by sampling rows and performing type inference.

Minimal scaffold created by the assistant. Use the CLI like:

```
cargo run -- -f examples/mixed_types.csv --dialect postgres
```

The project is a work-in-progress. The assistant scaffolded initial inference and DDL generation logic.
